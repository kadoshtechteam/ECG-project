# Dataset Folder

Place large ECG datasets in this folder locally, but do not commit them to Git.

Expected local files:
- `dataset/mitbih_train.csv`
- `dataset/mitbih_test.csv`
- `dataset/ptbdb_normal.csv`
- `dataset/ptbdb_abnormal.csv`

This keeps the project code and notebook paths stable while isolating dataset files from the repository.
